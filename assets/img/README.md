# /assets/img/

Drop product photography, hero images, OG images, and gallery photos here.

Suggested files:
- `og.jpg` (1200x630) - Open Graph share image referenced in every page <head>.
- `hero.jpg` - Replaces the CSS-composed hero scene on the homepage.
- `gallery/*.jpg` - Real project photos for the gallery page.
- `favicon.svg` - Already in place; replace if you want a different favicon.
